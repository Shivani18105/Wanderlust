# Wanderlust 🌍

A full-stack travel listing web application built using Node.js, Express, MongoDB, and EJS.

## Features
- User authentication
- Add/Edit/Delete listings
- Image uploads with Cloudinary
- Reviews and ratings
- Interactive maps
- Search functionality

## Live Demo
🔗 https://wanderlust-1-r32s.onrender.com/listings

## Tech Stack
- Node.js
- Express.js
- MongoDB Atlas
- EJS
- Bootstrap
- Cloudinary
